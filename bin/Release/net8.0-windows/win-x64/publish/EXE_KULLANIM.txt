========================================
  CAKA Personel Takip - EXE Kullanım
========================================

Tüm "publish" klasörünü (içindeki EXE + tüm dosya ve klasörlerle birlikte) olduğu gibi kopyalayın. Sadece EXE'yi alırsanız program çalışmaz; yanındaki dosyalar da gerekli.

  1. Klasördeki "Caka Personel Takip.exe" dosyasına çift tıklayın.
  2. İnternet bağlantısı gerekir.
  3. Varsayılan API adresi programın içinde (https://caka-api.onrender.com).

API adresini değiştirmek isterseniz (isteğe bağlı):
  Kullanıcı klasöründe AppData\Local\CAKA\ içine CAKA.config.json oluşturun:
  {"ApiBaseUrl":"https://your-api.com","ApiTimeoutSeconds":30}

İlk giriş (yönetici):
  Kullanıcı adı: admin
  Şifre: 1234

.NET yüklemenize gerek yoktur; EXE kendi içinde çalışır.

--- "Akıllı Uygulama Denetimi" veya "Bilgisayarınızı korudu" engeli ---
  Uygulama imzalı olmadığı için Windows ilk seferde engelleyebilir. Aşağıdakilerden biriyle açabilirsiniz:

  YÖNTEM 1 (önerilen):
    1. Engel penceresinde "Diğer bilgiler" / "More info" linkine tıklayın.
    2. Altta çıkan "Yine de çalıştır" / "Run anyway" düğmesine tıklayın.
    → Program açılır; bir sonraki çalıştırmada aynı uyarı genelde çıkmaz.

  YÖNTEM 2:
    EXE'ye sağ tık → Özellikler → En altta "Bu dosyayı engellemeyi kaldır" (Unblock) kutusu varsa işaretleyin → Tamam. Sonra EXE'yi tekrar çalıştırın.

  YÖNTEM 3 (tüm bilinmeyen uygulamalar için):
    Windows 11: Ayarlar → Güncelleme ve güvenlik → Windows Güvenliği → Uygulama ve tarayıcı denetimi → "Akıllı Uygulama Denetimi"ni Kapalı yapın. (Sistem genelinde kapalı olur.)
